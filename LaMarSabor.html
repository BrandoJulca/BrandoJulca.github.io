Proyecto: La Mar de Sabor

Este proyecto demuestra un enfoque completo en el desarrollo web utilizando tecnologías Java, Maven y Tomcat para la parte del backend y alojamiento del sitio. Además, la implementación de hojas de estilo CSS contribuye a una mejor apreciación visual de la página web, brindando una experiencia más agradable para los usuarios.

1) Tecnologías utilizadas
- Java: Se ha utilizado el lenguaje de programación Java para desarrollar la lógica de negocio y la funcionalidad del sitio web.
- Maven se ha empleado como herramienta de gestión y construcción del proyecto, facilitando la administración de dependencias y la compilación.
- Tomcat: Para alojar la página web en un servidor local (localhost), se ha utilizado Apache Tomcat, que es un contenedor de servlets de código abierto.
- HTML y CSS: El diseño y la estructura visual de la página web se han creado utilizando HTML (HyperText Markup Language) y hojas de estilo CSS (Cascading Style Sheets).
- Base de datos: Es probable que el proyecto utilice una base de datos para almacenar información de los usuarios, pedidos y otros datos relevantes.

2) Funcionalidades del sitio web
-Categorías de comida: Los usuarios pueden elegir entre diferentes categorías de comida, lo que sugiere que el sitio web ofrece una variedad de opciones gastronómicas.
-Selección de platillos: Una vez que se elige una categoría de comida, los usuarios pueden seleccionar el platillo específico que desean consumir.
-Número de platillos: Los usuarios pueden indicar la cantidad de platillos que desean pedir de un determinado tipo.
-Registro de usuario: Para realizar un pedido, el sitio web solicita un usuario y contraseña, lo que probablemente implica la creación de cuentas de usuario para facilitar futuras compras.
-Visualización del pedido: Después de realizar el pedido, el sitio web permite a los usuarios visualizar su pedido de compra antes de confirmarlo.
-Cálculo del total a pagar: El sitio web realiza el cálculo del monto total a pagar, considerando la cantidad y el precio de los platillos seleccionados.
-Generación de voucher: Una vez confirmado el pedido, el sitio web genera un voucher (comprobante) con los detalles del pedido y el total a pagar, que se puede imprimir.
-Este proyecto demuestra un enfoque completo en el desarrollo web utilizando tecnologías Java, Maven y Tomcat para la parte del backend y alojamiento del sitio. Además, la implementación de hojas de estilo CSS contribuye a una mejor apreciación visual de la página web, brindando una experiencia más agradable para los usuarios.
